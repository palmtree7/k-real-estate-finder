# K-Real Estate Finder 🏠

> 수도권에서 **진짜 믿을 수 있는 부동산**을 찾고 계신가요?

오래된 부동산일수록 그 지역에서 신뢰를 쌓아온 곳입니다.
K-Real Estate Finder는 한국공인중개사협회 공식 데이터를 기반으로
수도권 지역별 **베테랑 중개사**를 한눈에 찾아드립니다.

## ✨ 왜 K-Real Estate Finder인가?

- 협회 대표 의원급 중개사만 수록
- 지역별 빠른 검색
- 공식 협회 데이터 기반으로 신뢰도 높음

## 🔍 사용법
1. 지역 선택 
2. 검색
3. 연락처 바로 확인

## 🛠 기술 스택
- Python / Flask
- Playwright / BeautifulSoup
